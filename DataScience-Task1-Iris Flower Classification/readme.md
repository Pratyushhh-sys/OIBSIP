# 🌸 Iris Flower Classification using Machine Learning

## 📌 Project Overview

This project is part of the **Oasis Infobyte Data Science Internship (OIBSIP)**.

The objective is to build a Machine Learning classification model that predicts the species of an Iris flower using its physical measurements:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

The project includes data exploration, visualization, model training, performance evaluation, and comparison of multiple machine learning algorithms.

---

## 🎯 Objective

- Load the Iris dataset from Scikit-learn
- Perform Exploratory Data Analysis (EDA)
- Visualize feature distributions and relationships
- Train multiple classification models
- Compare model performance
- Identify the best-performing model

---

## 📂 Dataset

The project uses the **Iris Dataset** available directly in **Scikit-learn**.

Dataset Information:
- Total Samples: **150**
- Features: **4**
- Classes: **3**

Species:
- Iris Setosa
- Iris Versicolor
- Iris Virginica

---

## 📊 Exploratory Data Analysis (EDA)

The following analyses were performed:

- Dataset Overview
- Missing Value Check
- Statistical Summary
- Pair Plot

- Correlation Heatmap
- Species Distribution Count Plot

## 🤖 Machine Learning Models Used

The following classification algorithms were implemented:

- Logistic Regression
- K-Nearest Neighbors (KNN)
- Decision Tree Classifier
- Random Forest Classifier

---

## 📈 Evaluation Metrics

The models were evaluated using:

- Accuracy Score
- Confusion Matrix
- Classification Report
- Model Performance Comparison

---

## 🏆 Results

The models achieved excellent classification performance on the Iris dataset.

Best-performing models achieved **100% accuracy** on the test dataset.

The analysis also showed that:

- Petal Length
- Petal Width

are the most important features for predicting Iris species.
