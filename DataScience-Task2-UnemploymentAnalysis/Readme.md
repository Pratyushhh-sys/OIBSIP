# 📊 Unemployment Analysis with Python

## 📌 Project Overview

This project is part of the **Oasis Infobyte Data Science Internship (OIBSIP)**.

The objective of this project is to analyze unemployment trends in India using Python and Exploratory Data Analysis (EDA). The analysis focuses on identifying regional and temporal unemployment patterns, understanding the impact of the COVID-19 pandemic, and generating meaningful insights through data visualization.

---

## 🎯 Objective

- Load and explore the unemployment dataset.
- Perform data cleaning and preprocessing.
- Analyze unemployment trends across different states and regions.
- Study monthly unemployment patterns.
- Compare unemployment rates before and after COVID-19.
- Generate visualizations and derive meaningful insights.

---

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook / Google Colab

---

## 📂 Dataset

**Dataset:** Unemployment in India

The dataset contains unemployment statistics across Indian states and union territories, including:

- Region
- Date
- Frequency
- Estimated Unemployment Rate (%)
- Estimated Employed
- Estimated Labour Participation Rate (%)

---

## 📊 Project Workflow

### 1. Data Loading
- Imported the dataset using Pandas.
- Inspected dataset structure and columns.

### 2. Data Cleaning
- Renamed columns for better readability.
- Converted the date column to datetime format.
- Removed duplicate records.
- Handled missing values.

### 3. Exploratory Data Analysis (EDA)

Performed the following analyses:

- Dataset overview
- Missing value analysis
- Statistical summary
- Distribution of unemployment rates
- Region-wise average unemployment
- State-wise unemployment comparison
- Monthly unemployment trends
- Time-series analysis for major states
- Correlation heatmap
- COVID-19 impact analysis

---

## 📈 Visualizations

The notebook includes:

- Histogram of unemployment rates
- Region-wise bar chart
- State-wise bar charts
- Monthly trend line chart
- Time-series plots for selected states
- Correlation heatmap
- Pre-COVID vs Post-COVID comparison chart

---

## 🔍 Key Insights

- Significant variation exists in unemployment rates across Indian states.
- Certain regions consistently recorded higher unemployment levels.
- Monthly trend analysis highlighted fluctuations over time.
- The COVID-19 pandemic caused a noticeable increase in unemployment rates.
- Labour participation and employment indicators provide additional context for understanding unemployment trends.
